### 修复记录

测试过程中发现供应商配置有误并已修复：

| 字段 | 修改前 | 修改后 | 原因 |
|------|-------|-------|------|
| code | openai | anthropic | API 实际走 Anthropic 协议 |
| provider_type | openai_compatible | anthropic | 否则工厂会创建 OpenAIProvider 调用错误的端点 |
| base_url | .../coding/v3 | .../coding/anthropic/v1 | AnthropicProvider 拼接 /messages，需要正确的基础路径 |
