# EasyMVP V3 审计列表线框图设计

> 更新时间：2026-04-19
> 上游文档：[EasyMVP-V3-审计列表页面设计](./EasyMVP-V3-审计列表页面设计.md)
> 关联文档：[EasyMVP-V3-审计过滤器设计](./EasyMVP-V3-审计过滤器设计.md)
> 目标：把审计列表页推进到线框图级别，明确过滤区、列表区、详情区的版式。

## 1. 线框结构

```text
┌────────────────────────────────────────────────────────────┐
│ Filters                                                   │
├───────────────────────────────┬────────────────────────────┤
│ Audit Record List             │ Audit Detail              │
│ time / kind / summary         │ selected record info      │
└───────────────────────────────┴────────────────────────────┘
```

## 2. 关键规则

1. 过滤器固定在顶部
2. 列表支持快速扫读
3. 详情区可打开 replay

## 3. 不该怎么做

不应该：

1. 只有表格没有详情区
2. 详情必须跳转新页面
